# Shutter Test
2011년에 제작된 셔터 테스트 프로그램.  
도가니의 View Port 를 자동으로 여닫으며 설치된 카메라를 통해 출력되는 외부 프로그램과 함께 스크린샷을 남긴다.  
셔터 설정등은 지정된 Recipe 를 통해 변경 가능.

## 사용기술 및 환경
* Delphi2007
* I700 Series 통신 모듈
